# Real Example Runs

Short, public-facing examples derived from real Claude + Codex debate runs.

These are not full raw logs. Each example is edited for readability and focuses on:

- the decision setup
- the strongest challenge from the opposing model
- the final synthesis

Raw unedited logs are kept locally in each project's `.claude/debate-logs/` folder and are not committed. Available on request.

Current example:

- `transcription-stack-direction.md` — choosing between a pure-local, hybrid, or hosted direction for a local-first transcription pipeline
